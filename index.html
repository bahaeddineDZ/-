هل شاهدت هذا الفيلم؟ — Did You Watch This Movie?

🇸🇦 وصف المشروع (بالعربية):

تطبيق ويب بسيط يعرض للمستخدم بوسترات لأفلام عشوائية من منصة TMDB ويطلب منه فقط اختيار ما إذا شاهد الفيلم أم لا. الهدف هو مساعدة المستخدم على تتبع الأفلام التي شاهدها، مع عرض إحصائيات بسيطة حول نشاطه السينمائي. التطبيق غير تجاري ومخصص للاستخدام الشخصي والتجريبي فقط.

🇺🇸 Project Description (English):

A minimalistic web application that shows a random movie poster from TMDB and asks the user if they have watched it (Yes/No). The goal is to help users track which movies they have seen and generate simple viewing statistics for personal use. No commercial purpose. Built with HTML and JavaScript using TMDB API for educational and personal tracking purposes.
